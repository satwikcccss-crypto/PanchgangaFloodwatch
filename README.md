# PanchgangaFloodwatch — RTDAS Dashboard

**Owner:** Centre for Climate Change and Sustainability Studies (CCCSS), Shivaji University, Kolhapur  
**Status:** 🚀 **SCADA Redesign Applied (April 2026)**

---

## Technical Overview
The system has been upgraded to a high-fidelity **SCADA** command center.

### Recent Changes
- **RWL Nomenclature**: Global shift to Radar Water Level (RWL) terminology.
- **Metric Standardization**: All telemetry now monitors in **Meters (m MSL)**.
- **SVG Telemetry**: Integrated animated SVG engineering gauges.
- **UI Optimization**: Stripped redundant statistical bars and legends for a GIS-centric view.
- **WSL Analytics**: Renamed analytical views to Water Surface Elevation (WSL).
